# AssemblyProject

AssemblyProject is a Spring Boot application that interacts with a Redis database to store and retrieve data.

## Prerequisites

Before running this application, ensure you have the following installed:

- JDK 17 or higher
- Maven
- Docker (if running Redis in a Docker container)

## Setup

1. Clone this repository to your local machine.
2. Navigate to the project directory.

## Running the Application

### Without Docker:

1. Make sure Redis is running on your local machine.
2. Build the project using Maven:
3. Run the application:
4. The application will be accessible at `http://localhost:8080`.

### With Docker:

1. Make sure Docker is installed on your local machine.
2. Build the Docker image:
3. Run Redis in a Docker container:
4. Run the Spring Boot application in a Docker container:
5. The application will be accessible at `http://localhost:8080`.

## Usage

1. Use Postman or any other HTTP client to interact with the application.
2. Send requests to `http://localhost:8080` to store and retrieve data from Redis.

## Additional Notes

- Make sure to update the application configuration (`application.properties` or `application.yml`) with the correct Redis host and port if necessary.
- For more advanced configurations, refer to the Spring Boot documentation.
